# AgriGuru

AgriGuru is an AI-powered agricultural platform designed to empower farmers with localized recommendations, disease detection, weather forecasts, and a knowledge hub.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`

2. Environment Variables:
   Set your `GEMINI_API_KEY` in `.env.local` for the AI features to work.

3. Run the application:
   Start the frontend:
   `npm run dev`
   
   Start the backend server (if required):
   `npm run server`
